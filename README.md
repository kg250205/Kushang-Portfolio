# Kushang Goel - Personal Portfolio

A modern, high-performance, and visually stunning Single-Page Application (SPA) portfolio showcasing my journey as a Data & Business Analyst and Software Tester.

## 🚀 Live Demo

Explore the live site: [https://kg250205.github.io/Kushang-Portfolio/](https://kg250205.github.io/Kushang-Portfolio/)

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (Utility-first framework)
- **Styling & Effects**: Glassmorphism, CSS Custom Properties, smooth hover animations, and Font Awesome Icons.
- **Tools**: Streamlit (utilized in featured projects), Git, GitHub.

## 📁 Key Sections

1. **Summary / Profile**: A concise overview of my academic background (pursuing MCA in AI & ML, BCA graduate) and technical competencies.
2. **My Expertise**:
   - Data & Business Analytics

   - Artificial Intelligence & Machine Learning
   - Computer Science & Information Technology
3. **Education**: Master of Computer Applications (MCA) and Bachelor of Computer Applications (BCA).

5. **My Journey**:
   - Detailed Work Experience timeline.

   - Product Recommendation System

7. **Contact**: Centered contact interface featuring Email, WhatsApp, LinkedIn, and GitHub links.

## 🔧 Getting Started

To view or run the portfolio locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/kg250205/Kushang-Portfolio.git
   ```
2. Open `index.html` in any modern web browser. No compilation or package installation is required!
